Class Section

A Section object is used to encode values organized by Field objects within a Format object. A single Format encodes mulpiple Field objects, which are assigned values inside a section object.

A Section object is what you send through yodel, and what you will receive. (For smaller projects you can also send and receive unformatted strings).

fields

fields holds the values assigned to the different Field objects in Section.format. They are organized like this:

{
     "fieldname": fieldValue
}

For each field lined up in Section.format.

payload: string = ""

In cases where yodel is being used to send raw strings outside of Section objects, payload will contain the raw string on the receiving end. Otherwise, payload contains any extra bytes that could not be assigned to fields.
